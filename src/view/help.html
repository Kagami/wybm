Help

wybm is a GUI application which can download videos from YouTube in WebM format and interactively cut them without reencoding. Its main purpose is to preserve original quality of the YouTube clip while still being able to do few simple manipulations.

Fragment start

To mark start of the fragment, press button. By default it points to the start of the file. Note: fragment can start only at keyframe. You can navigate between keyframes with keyboard arrow keys.

Fragment end

To mark end of the fragment, press button. By default it points to the end of the file. Fragment can end at any point but not before the fragment start.

Preview

Some imageboards use only first single frame of the video in order to generate thumbnail. It maybe be undesirable in some cases, e.g. a lot of clips start with a black frame. wybm allows you to overcome with limitation and select custom preview either from file or using current video frame. Note: technically it works by creating additional video track with higher resolution, beware that some sites may forbid videos with more than one video track. By default preview is not created.

Size

wybm shows you approximate size of the resulting file and updates this number when you move the fragment boundaries. You can also press button in order to fit fragment end to the desired limit while keeping fragment start intact. Note that most the time resulting file will be slighly (100-200KiB) bigger. (Also note that there are 1024 bytes in kibibyte.)

Save

When you are done with fragment boundaries and preview, simply press button in order to generate resulting file. It should be very fast and you can also return to the exact current view later, so don't hesitate trying different variants. With default fragment boundaries it will simply copy entire video to the given destination.

SPACEToggle pause

DOTGo to next frame
